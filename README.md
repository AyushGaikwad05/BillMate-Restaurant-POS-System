# 🍽️ BillMate – Restaurant POS System

BillMate is a modern and simple **Restaurant POS System** built using **Next.js**, **Redux Toolkit**, and **TailwindCSS**.  
It helps restaurant staff manage **customers, tables, and orders** in a fast, organized, and efficient workflow.

---

## 🚀 Features

### ✅ Customer Management
- Add customer name  
- Add phone number  
- Select number of guests  
- Global state stored using Redux  
- Fast and smooth POS workflow  

### ✅ Table Management
- View table availability (Available / Booked)  
- Assign customers to tables  
- Smart redirection to menu/order interface  
- Essential for restaurant POS operations  

### ✅ Order Management
- Create and store customer orders  
- Connected restaurant POS workflow  
- Ready for menu, billing, and invoicing expansions  

### ✅ Modern UI
- Clean bottom navigation (Home, Orders, Tables, More)  
- Beautiful modal for order creation  
- Responsive UI suitable for tablets used in restaurants

## 📸 Core POS Workflow

### 1️⃣ Add Customer  
Staff enters:
- Customer Name  
- Phone Number  
- Guest Count  

### 2️⃣ BillMate Saves Data in Redux  
Stored globally for use across pages.

### 3️⃣ Select a Table  
Staff selects an available table to assign the customer.

### 4️⃣ Redirect to Menu Page  
Staff can now start taking orders.

## ⚙️ Installation

### 1. Clone the Repository
```bash
git clone <your-repo-url>
cd billmate

```
### 2. Install Dependencies
npm install

### 3. Start Development Server
npm run dev


Your Restaurant POS System will be running at:
👉 http://localhost:3000
